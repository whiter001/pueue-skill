# pueue-skill

管理后台任务的 AI Skill，用于与 [pueue](https://github.com/nickelpueue/pueue) 任务队列管理工具交互。

## 功能

当用户想要在后台运行命令、管理任务队列、查看任务状态或控制任务执行时使用此 skill。

### 核心功能

- **添加任务**: 在后台执行命令，支持延迟、标签、分组
- **查看状态**: 显示所有任务和分组的当前状态
- **查看日志**: 查看任务的标准输出和错误
- **控制执行**: 暂停、恢复、终止、重启任务
- **清理**: 清理已完成的任务
- **分组管理**: 创建分组、设置并行任务数

## 使用方式

当用户说以下内容时调用此 skill：
- "在后台运行..."
- "异步执行..."
- "查看任务状态"
- "管理任务队列"
- "后台执行..."
